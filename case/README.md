## 💡 关键要点

### 问题
支付 API 返回 401 签名错误，导致支付流程失败

### 根本原因
Authorization header 中使用了错误的 ID

### 解决方案
1. 使用正确的证书序列号
2. 添加缺失的 HTTP header
3. 改用正确的支付模式
4. 添加必需的参数

### 结果
✅ 支付流程正常工作
